# 题目描述
给你一个整型数组 nums ，在数组中找出由三个数组成的最大乘积，并输出这个乘积。

# 实例


输入：nums = [1,2,3]      
输出：6           

输入：nums = [1,2,3,4]           
输出：24        
     
输入：nums = [-1,-2,-3]     
输出：-6     


# 解题方法


```java
class Solution {
    public int maximumProduct(int[] nums) {
        int n = nums.length;
        Arrays.sort(nums);
        return Math.max(nums[0] * nums[1] * nums[n - 1], nums[n - 1] * nums[n -2] * nums[n - 3]);
    }
}
```
